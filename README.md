# HOPE Hybrid Cycle Explorer

A working thermodynamic exploration environment for the **HOPE Engine Cycle**.

This tool allows engineers and researchers to evaluate the behavior of the HOPE hybrid thermodynamic cycle through an interactive model interface. The explorer computes cycle performance, energy distribution, and system behavior based on adjustable input parameters.

The goal is not to present simplified charts but to allow direct engineering evaluation of the model.

---

## Purpose

The HOPE Explorer is designed for:

• Engineers evaluating thermodynamic cycle performance  
• Researchers studying alternative high-efficiency engine cycles  
• OEM developers exploring next-generation power systems  
• Policy and sustainability analysts studying energy conversion efficiency  

The explorer allows users to directly inspect system outputs rather than relying on presentations or theoretical claims.

---

## Features

• Interactive thermodynamic model interface  
• Performance graph generation  
• Energy partition Sankey visualization  
• Internal Heat Recovery Loop (IHRL) flow representation  
• CSV export of calculated results  
• PNG export of generated graphs  
• PDF report generation  

---

## Technology Stack

Framework: **Next.js**  
Language: **TypeScript**  
Visualization: **Recharts**  
Export Tools: **html-to-image**, **jsPDF**
